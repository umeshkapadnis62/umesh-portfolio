## 📌 About This Project

This is my **personal portfolio website** built using **HTML, CSS, and JavaScript**.
It showcases my skills, projects, and education as an aspiring **Java Full Stack Developer**.

The portfolio features a **modern neon UI**, smooth animations, and interactive elements to create a professional user experience.

---

## ✨ Features

* ⚡ Modern **Neon Glassmorphism UI**
* 🎯 Fully **Responsive Design** (Mobile + Desktop)
* 🌙 **Dark/Light Mode Toggle**
* 🎬 Smooth **Scroll Animations**
* 💡 Interactive **Skill Progress Bars**
* 🚀 **Project Showcase Section**
* 📩 Contact Form (Frontend Ready)
* 🎨 Particle Background Effects
* 📊 Scroll Progress Indicator

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Styling:** Custom CSS (Glassmorphism + Neon Effects)
* **Icons:** Font Awesome
* **Fonts:** Google Fonts (Inter)
